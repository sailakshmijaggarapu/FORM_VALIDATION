# Contact Form with Validation

## Overview

This project is a responsive Contact Form developed using HTML, CSS, and JavaScript. The form collects user information such as Name, Email, Phone Number, and Message. JavaScript validation ensures that all fields are correctly filled before submission.

## Features

* Responsive and user-friendly design
* Modern CSS styling
* Name validation (letters and spaces only)
* Email format validation
* Phone number validation (exactly 10 digits)
* Message validation
* Error and success messages
* Interactive button hover effects

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Form Fields

* Name
* Email
* Phone Number
* Message

## Validation Rules

* All fields are required.
* Name should contain only alphabets and spaces.
* Email should be in a valid format.
* Phone number must contain exactly 10 digits.
* Message should contain at least 5 characters.

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in a web browser.
4. Alternatively, use the Live Server extension in VS Code.

## Expected Output

* Displays an error message if any validation rule is violated.
* Displays a success message when all inputs are valid.
